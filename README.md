Деврел-дашборд — удобный, масштабируемый, кастомизруемый UI для вашего деврельского дашборда (или трекинга BHT)
Я делаю его для себя, но так, чтобы его смог переиспользовать любой желающий, без привязки к конкретной компании и с возможностью подключения своих библиотек или БД. 

**Функции**
- Агрегация данных из любой подключенной БД (в т.ч. Google Spreadsheet как источник данных)
- Удобная no-code настройка отображаемых графиков (на базе Apache SuperSet)
- Возможность докрутить несложную бизнес-логику: например, красить графики по достижению целевых значений
- Экспорт в PDF, CSV и пуш-уведомления в ТГ (личным сообщением от бота, либо в выбранный канал/чат)

**Основные метрики**
1. Узнаваемость бренда (Brand Awareness)
Спонтанная и подсказанная узнаваемость: частота упоминания компании без подсказки и узнаваемость среди списка брендов.
Трафик по брендовым поисковым запросам.
Посещаемость сайта без реферальных ссылок.

2. Имиджевые ассоциации и восприятие
Восприятие бренда аудиторией, ключевые ассоциации (например, инновационность, надежность, экспертность в IT).
Оценка через регулярные опросы и анализ комментариев в социальных сетях.

3. Сентимент и тональность (Brand Sentiment)
Процент положительных, нейтральных и отрицательных упоминаний о компании.
Динамика изменения настроения аудитории.
Анализ эмоционального фона в соцсетях, отзывах, комментариях.

4. Лояльность и удовлетворенность
NPS (Net Promoter Score): готовность рекомендовать компанию коллегам и партнерам.
Уровень удовлетворенности (CSI).
Анализ отзывов.

5. Репутация и отзывы
Системный мониторинг отзывов на профильных платформах.
Анализ реакций на кризисные ситуации и изменения.
Анализ причин негативных и позитивных откликов.

6. Динамика бизнес-показателей
Рост выручки, новых клиентов, повторных обращений.
Доля рынка по целевым направлениям.
Конверсия в целевые действия.
